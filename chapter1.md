# 安裝

## 開發工具

1. Visual Studio Code
2. Node JS
3. Angular CLI
4. Firebase tool

## 安裝方式

1. Visual Studio Code
   1. 網站下載, [https://code.visualstudio.com/download](https://code.visualstudio.com/download)
2. Node JS
   1. 網站下載, [https://nodejs.org/](https://nodejs.org/)
   2. 請安裝 Version 6
   3. 內附npm, 更新版本

   ```
   >npm i -g npm
   ```
3. Angular CLI  
   1. 使用npm安裝

   ```
   >npm i -g @angular/cli
   ```

4. Firebase tool  
   1. 使用npm安裝

   ```
   >npm install -g firebase-tools
   ```
